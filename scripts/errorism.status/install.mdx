## ขั้นตอนการติดตั้ง

โปรดตรวจสอบขั้นตอนทั้งหมดในการติดตั้งนี้ก่อนที่คุณจะเปิด Ticket ในชุมชนของเรา เนื่องจากส่วนใหญ่ของ Ticket มาจาการละเว้นขั้นตอนบางอย่างในคู่มือนี้ แต่ละขั้นตอนนั้นสำคัญ

<Card title="ดาวน์โหลดทรัพยากร" icon="square-1" color="#ea5a0c">
  คุณจะต้องดาวน์โหลดทรัพยากรใหม่ที่คุณซื้อมาก่อนหน้านี้จาก script.errorism.cc
  สำหรับสิ่งนี้เราต้องเข้าสู่ระบบด้วยบัญชีของเราที่ keymaster และค้นหาส่วนของ
  Asset grants ที่นั่นเราจะพบแพ็คเกจของเราที่สั่งซื้อไปแล้ว หรือจะดาวน์โหลดที่
  email ที่คุณได้รับจาก tebex ก็ได้

<Card title="การติดตั้งที่ต้องใช้" icon="square-2" color="#0285c7">
  โปรดอย่าข้ามขั้นตอนการติดตั้งที่ต้องใช้ (dependencies) ใด ๆ
  ที่ระบุไว้ด้านล่างนี้ และหากคุณมีการติดตั้งที่ต้องใช้อยู่แล้ว
  โปรดทำการติดตั้งใหม่ตามเวอร์ชันที่แนะนำด้านล่างนี้
</Card>

| Resource   | Version    | Link                                                                   |
| ---------- | ---------- | ---------------------------------------------------------------------- |
| `ox_lib`   | **3.6.0+** | https://github.com/overextended/ox_lib/releases                        |
| `artifact` | **6497+**  | https://runtime.fivem.net/artifacts/fivem/build_server_windows/master/ |

<Card title="การวางตำแหน่งทรัพยากร" icon="square-3" color="#16a34a">
  หลังจากดาวน์โหลดทรัพยากรแล้ว ให้คุณทำการแตกไฟล์ที่คุณดาวน์โหลดมา
  และนำไปวางไว้ที่ `resources` ในโฟลเดอร์ของ FiveM Server
  ของคุณและเรียงลำดับตามดังนี้
  ```cfg
  ensure ox_lib
  ensure errorism.status
  ```

<Check>
  ทรัพยากรนี้พร้อมใช้งานเรียบร้อยแล้ว
  หากเกิดปัญหากรุณาทำตามขั้นตอนในการติดตั้งอีกครั้ง หรือติดต่อเราผ่านทาง Ticket
  ในชุมชนของเรา
</Check>
